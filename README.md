# Aphidex-Data

**EN:** Community-driven data files for the Aphidex app (offline updates via Google Play).  
**ES:** Archivos de datos comunitarios para la app Aphidex (actualizaciones offline vía Google Play).

## What is this?
**EN:** This repository stores the JSON data used by Aphidex. The app remains **fully offline**. Updates are shipped through **Google Play updates** only.  
**ES:** Este repositorio guarda los JSON usados por Aphidex. La app se mantiene **totalmente offline**. Las actualizaciones llegan **solo** mediante actualizaciones en Google Play.

## Contributing / Cómo contribuir
- ✅ Open an Issue to report errors or missing info.
- ✅ Open a Pull Request to propose changes to the JSON.
